📝 Task Manager com Sistema de Prioridades

Este projeto é um gerenciador de tarefas (To-Do List) desenvolvido com HTML, CSS e JavaScript puro, com foco em organização de tarefas por prioridade e persistência de dados no navegador.

O objetivo do projeto foi praticar conceitos fundamentais de desenvolvimento front-end, manipulação do DOM, lógica em JavaScript e organização de interface.

Este projeto faz parte do meu processo de aprendizado em desenvolvimento de software enquanto curso Análise e Desenvolvimento de Sistemas.

🚀 Funcionalidades

✔️ Adicionar tarefas
✔️ Escolher prioridade da tarefa (Urgente, Média ou Baixa)
✔️ Organização automática das tarefas por prioridade
✔️ Exclusão de tarefas
✔️ Salvamento automático das tarefas no navegador utilizando LocalStorage
✔️ Interface simples e organizada para visualização das tarefas
✔️ Modal para seleção de prioridade antes de criar a tarefa

💾 Persistência de dados

As tarefas são armazenadas utilizando:

localStorage

Isso permite que as tarefas continuem salvas mesmo após atualizar a página ou fechar o navegador.
